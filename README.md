# Netflix Clone Web App

A responsive **Netflix-inspired web page** built using **HTML, CSS, and JavaScript**. This project showcases modern frontend skills including responsive layouts, dynamic form validation, and interactive UI elements.  

Live Demo: [Netflix Clone](https://aceraspire5121-debug.github.io/Netflix--Clone/)  

---

## Features

- **Responsive Design**  
  - Works seamlessly on desktop, tablet, and mobile devices.  
  - Dynamic scaling for fonts, buttons, and input fields using CSS `clamp()` and media queries.  

- **Landing Page with Hero Section**  
  - Full-screen background image with overlay for better readability.  
  - Prominent call-to-action buttons for user interaction.  
  - Email input with **real-time validation** and alerts.  

- **Content Sections**  
  - Grid-based layout for movie/show cards.  
  - Hover animations with smooth scaling and shadow effects for interactivity.  
  - Flexible grid that adapts to different screen sizes.  

- **FAQ Section**  
  - Accordion-style questions that highlight on hover.  

- **Footer Links**  
  - Organized into multiple columns, responsive for smaller screens.  

- **Form Validation**  
  - Validates email inputs with regex before alerting the user.  
  - Handles empty input and invalid email formats.
